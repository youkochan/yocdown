# YocDown

YocDown 是一个 sublime 插件，用于解决多种需求下的 markdown 高亮问题。

配色方案暂时基于 `Monokai Extended`，暂时在此基础上做了一些简单的修改，以后会进行更多个性化定制以及各种自动补全机制等。

# 平台

本插件仅在 `OS X` 以及 `Windows` 下进行测试。


# 更新日志

* 2016-01-04：最初版本，解决了最基本的语法高亮问题。
